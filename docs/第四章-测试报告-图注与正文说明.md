# 第四章 测试报告图注与正文说明

## 图4-1 测试运行结果

**图注：**  
图4-1 Hypium 测试框架运行结果

**正文说明：**  
图4-1 展示了 HarmonyExpense 项目在 Hypium 测试框架下的核心测试运行结果。测试覆盖用户数据访问、账单数据访问、AES-256-GCM 加密导出、云端双向增量同步以及 WAL 模式下的批量写入等关键模块，共执行 5 组测试套件、42 个测试用例，结果均通过。测试输出表明，项目在数据一致性、事务正确性、加密校验、同步幂等性和高并发批处理等方面均达到了预期目标，说明系统核心业务逻辑具备较好的稳定性与可验证性。

## 图4-2 性能监控报告

**图注：**  
图4-2 PerformanceMonitor 性能监控报告

**正文说明：**  
图4-2 给出了 HarmonyExpense 在典型高负载场景下的性能监控结果。报告重点展示了 10,000 条账单批量插入、海量账单游标分页查询、5 MB 备份数据的 AES-256-GCM 加密以及 PBKDF2 100000 次密钥派生等核心操作的耗时与内存占用情况。结果显示，在 `BatchQueryHelper`、`CacheManager`、复合索引和 SQLite WAL 模式等优化措施支持下，批量写入与分页查询均保持毫秒级响应，说明系统在数据规模增长和复杂业务场景下仍具备较好的性能稳定性与工程可用性。
